# Game Description

Trapped in a room, unable to move, with a whiteboard in the room. You Did This.

Not really a game but a concept that if someone cheated in one of my game, it would force close the game and open this.

This non game exist because of a real whiteboard drawing I made in high school. It was on there for about 130 days. (Roughly 4 mouths) Before finally being erased.

For more information press "H" in the game.
